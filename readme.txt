朋友圈一直被一些恶俗的测试刷屏
测测他是你的真命天子吗？
测测你是个合格的狮子男吗？
测测你最近的桃花运如何？
测试题目是一道道可以引起你展现欲的伪哲理问题：
比起高楼竖立的都市你更喜欢宁静的田园？
你相信有柏拉图式的爱情吗？
你绝对不吃不喜欢的食物？
而测试的结果则是一段段模棱两可，瞎子算卦式的屁话：
你的内心远比你的外表更渴望宁静；
你是一个忠于自己内心的人;
你是一个性格鲜明的人;
诸如此类恶俗不堪，妄图以星座/运势引起你共鸣的测试，数见不鲜。
于是写了这个小程序，并安利给了朋友圈的某些同学，想借此把那些脑残的测试批判一番。
结果呢，效果还不错，不少同学都信以为真了。
